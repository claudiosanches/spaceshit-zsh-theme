# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-28

### Added

- Initial release of Spaceshit ZSH Theme.
- Compact path display rooted at the current Git repository.
- Git branch and status integration via Oh My Zsh helpers.
- Command duration timer with customizable threshold and precision.
- Visual feedback for command success/error (green/red arrow).
- Customizable colors for all prompt segments.
- Customizable Git status symbols via environment variables.
